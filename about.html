About Project

Agentic system for safe engagement, intelligence collection, and scam prevention.

Our system uses Supabase Authentication with OAuth support for both Google and GitHub. When a user initiates sign-in or sign-up, a modal-based interface is displayed, allowing authentication via trusted identity providers. Supabase securely handles OAuth validation, session creation, and token management. Once authenticated, the user is granted access to protected resources such as the honeypot dashboard, ensuring a seamless and secure authentication workflow.

  • Safety-first engagement rules
  • Actionable intelligence output
